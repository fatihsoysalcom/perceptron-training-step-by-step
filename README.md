# Perceptron Training Step-by-Step

This example demonstrates the core concepts of neural network training: forward propagation, loss calculation, and backpropagation. It implements a simple single-neuron perceptron from scratch, using only Python's standard library. The script prints step-by-step calculations for each training iteration, allowing users to "monitor" how the network learns and adjusts its weights and bias.

## Language

`python`

## How to Run

1. Save the code as `main.py`.
2. Run from your terminal: `python main.py`
